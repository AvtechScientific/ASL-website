Aerodynamics of a locomotive in a tunnel

Description

Simulation of a locomotive running through a tunnel. In this benchmark air flow velocity and pressure fields were computed.

Source code

locomotive_in_tunnel.cc

Platform used

  • CPU: AMD FX(tm)-6300 Six-Core Processor 3.5GHz
  • RAM: 8Gb
  • GPU: Sapphire AMD Radeon HD 7950 3GB GDDR5 Boost OC
  • OS: 64-bit Debian GNU/Linux

Numerical setup

  • Domain size: 187 x 125 x 500
  • Number of points: 11’687’500
  • Resolution: 0.08 m
  • Physical size: 14.96 x 10 x 40 m³
  • Number of iterations: 200’000
  • Time step: 4.8e-4 s
  • Physical time: 96 s

Results

  • Computation time: 4191 s
