Detailed Description

template<typename T>
class asl::Parameter< T >

This class stores parameter's value and the information needed to extract it from command line and/or configuration file

Examples:
cubeGravity.cc, cubePoroelasticGravity.cc, jumpingBox.cc, levelSetBasic.cc, levelSetFacetedGrowth.cc, levelSetNormalGrowth.cc, and poroelastic.cc.
