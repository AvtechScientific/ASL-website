Detailed Description

This class automatically accomodates newly created Parameters and then loads them

Examples:
acousticWaves.cc, cubeGravity.cc, cubePoroelasticGravity.cc, jumpingBox.cc, levelSetBasic.cc, levelSetFacetedGrowth.cc, levelSetNormalGrowth.cc, multicomponent_flow.cc, multiphase_flow.cc, pitot_tube_ice.cc, and poroelastic.cc.
